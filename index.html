About me

Freelance Frontend Developer with 4+ months of experience

I started coding back when I was 13, using Youtube Python tutorials, later on I started taking online courses (mostly on Codecademy) on Front end Web Development, I learned to use HTML (semantic, forms, ARIA attributes), Advanced pure CSS (flexbox, grid, design theory, browser compatibility) and Javascript (loops, iterators, modifying DOM). I feel comfortable coding responsive, accessible and interactive layouts, also from solving problems using Javascript and I aim to learn frameworks like Bootstrap and React. I consider myself great at team-working and I like interacting with other people on the team. I am native in spanish and fluent on english (B1).
